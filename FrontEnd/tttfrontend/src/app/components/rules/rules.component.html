Rules

  • To win you must get three pieces in a row
  • A round starts when the first player's turn starts
  • A round ends when the last player's turn ends
  • Only one double block may be used per round
  • A double block must be declared before placing down a piece
  • A double block can be used if two or more players have the opportunity to win on their next turn
  • A double block can not be used to win
  • A double block must be used to block
  • Each player gets one double block per game
